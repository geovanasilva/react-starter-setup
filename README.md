## Local installation

**Prerequisites**:

- [NPM](https://docs.npmjs.com/getting-started/installing-node)

Clone repository:

```
git clone git@github.com:geovanasilva/react-starter-setup.git

cd react-starter-setup
```

Install npm dependencies:

```
npm install
```
